Executor Integration

Bridge status, transport mode, and full tool reference for all 30+ available APIs.

Executor WS Bridge — Offline

No active Roblox clients connected to the internal bridge.

WebSocket Transport
Preferred
Full-duplex, real-time. Commands sent instantly. Supported by most modern executors. Uses port 16384.
HTTP Polling Transport
Fallback
Polls every 100ms. Commands queue server-side — multiple concurrent tool calls are batched and delivered together on the next poll cycle.

Architecture Overview This application combines Roblox process management and custom developer APIs into a single workspace.
  1. Clients launched using saved cookies authenticate natively.
  2. Upon creation they bind to the internal socket to await Lua commands.
  3. All console events, script indexers, and spies route back to your agent synchronously.
